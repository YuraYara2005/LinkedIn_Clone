# LinkedIn Clone - Full Stack Professional Networking Platform

This project is a full-stack LinkedIn-inspired professional networking platform built using PHP (custom MVC architecture) and MySQL. 
The application replicates and extends core LinkedIn functionalities, allowing users to build professional profiles, grow their network, share content, explore job opportunities, and interact in a dynamic social environment.
It was designed and developed with scalability, modularity, and clean architecture principles in mind.
